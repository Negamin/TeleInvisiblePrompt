# Contribuindo para o Invisible Teleprompter

Obrigado por querer contribuir!

## Como contribuir

1. Fork este repositório
2. Crie uma branch para sua feature ou correção (`git checkout -b minha-feature`)
3. Faça suas alterações
4. Envie um pull request explicando as mudanças

## Boas práticas
- Escreva mensagens de commit claras
- Documente suas funções e código
- Teste antes de enviar

Fique à vontade para abrir issues com sugestões, bugs ou dúvidas!
